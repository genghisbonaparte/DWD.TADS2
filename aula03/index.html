Arrays / Objetos String

  • ValueOf()
    • - Retorna o valor primitivo do objeto
  • charAt()
    • - retorna uma String contend caracter de posição
  • concat(String, String, ..., n)
    • - este método retorna uma string contendo uma string + string n vezes
  • replace
    • - busca na string uma sub string que seja igual ao conteúdo
  • slice(ini, fim)
    • - retorna uma copia substring contendo os caracteres do inicio até o fim
  • split(separator, limit)
    • - retorna um obj. de array contendo a substring entre o saparator e o limite
  • substring(ini, fim)
    • - retorna o resulta da conversão do objeto que o chamou o método (inicio até fim)
  • toLowerCase()
    • - Converte as letras de uma string em minusculas
  • toUpperCase()
    • - Converte as letras de uma string em maiusculas

Arrays

  • concat()
    • - agrupa todos os elementos do array
  • pop()
    • - remove o ultimo elemeto do array
  • push()
    • - insere o elemento no final do vetor
  • shift()
    • - o primeiro elemento do array removido
  • unshift()
    • - adicionna um elemnto no inicio do array
  • reverse()
    • - inverte a ordem dos elementos do array
  • sort()
    • - ordena o array (desde que sejam uma String)
  • lenght()
    • - propriedade contém o tamanho do vetor
  • splice(indice, n° de elementos)
    • - remove os elementos apartir do idice especificado e no numero especificado
